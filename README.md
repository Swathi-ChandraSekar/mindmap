# Interactive Mindmap UI

This is an **interactive, data-driven Mindmap UI** built with **React**.  
It visualizes hierarchical data as a **vertical tree** and supports node interactions, searching, theme switching, and exporting the mindmap as an image.

The UI is generated entirely from **JSON data**.
